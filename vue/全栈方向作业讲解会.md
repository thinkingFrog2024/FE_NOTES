## html标签+dom结构

1. **文档结构标签**
   - `<html>`：根标签，包围整个 HTML 文档。
   - `<head>`：包含文档的元数据，如 `<title>`、`<link>`、`<script>`、`<style>` 等。
   - `<body>`：文档的主体，包含可见的页面内容。
2. **文本内容标签**
   - `<p>`：段落标签，用于文本段落。
   - `<h1>` 到 `<h6>`：标题标签，`<h1>` 是最高级别的标题，`<h6>` 是最低级别。
   - `<strong>`：强调文本，通常显示为加粗。
   - `<em>`：强调文本，通常显示为斜体。
   - `<mark>`：标记文本，通常显示为高亮。
   - `<small>`：小号字体。
   - `<del>`：删除线文本，表示文本已被删除。
   - `<ins>`：下划线文本，表示文本已被插入。
3. **链接和导航标签**
   - `<a>`：锚点标签，用于创建超链接。
   - `<nav>`：导航链接的容器。
4. **列表标签**
   - `<ul>`：无序列表。
   - `<ol>`：有序列表。
   - `<li>`：列表项。
5. **图像和多媒体标签**
   - `<img>`：图像标签，用于嵌入图片。
   - `<video>`：视频标签，用于嵌入视频内容。
   - `<audio>`：音频标签，用于嵌入音频内容。
   - `<source>`：定义多媒体资源。
   - `<track>`：定义字幕或其它类型的文本轨道。
6. **表格标签**
   - `<table>`：表格标签。
   - `<tr>`：表格行。
   - `<th>`：表头单元格。
   - `<td>`：表格单元格。
   - `<thead>`：表格头部。
   - `<tbody>`：表格主体。
   - `<tfoot>`：表格脚部。
7. **表单和输入标签**
   - `<form>`：表单标签，用于收集用户输入。
   - `<input>`：输入字段，类型可以是 text、password、radio、checkbox 等。
   - `<textarea>`：多行文本输入。
   - `<button>`：按钮。
   - `<label>`：标签，与 `<input>` 元素关联，提高可访问性。
   - `<select>`：下拉选择框。
   - `<option>`：下拉选择框中的选项。
8. **语义化标签**
   - `<header>`：页面或区段的头部。
   - `<footer>`：页面或区段的脚部。
   - `<section>`：文档中的一个独立部分。
   - `<article>`：独立的内容区块。
   - `<div>`：文档中的一个分区。
   - `<span>`：行内的分区。

语义化则是需要根据不同的内容选择不同的标签。其实一个div就可以应付绝大多数情况了，那么我们为什么要强调语义化？

* 提高可读性：将页面使用header footer这样的标签进行分区，有助于其他人阅读页面，项目是需要很多人一起合作的 你写的页面是要交给别人阅读的 通篇的div看着就很烦
  * 阅读别人的代码就是在浏览器里面的调试工具里面看 
* 搜索引擎优化（SEO）：搜索引擎使用页面的结构来确定内容的重要性和相关性。使用语义化的标签可以帮助搜索引擎更好地理解页面内容，从而提高页面在搜索结果中的排名。(搜索引擎优化)
* 兼容：语义化的HTML有助于确保内容在不同的浏览器和设备上以一致的方式显示，因为这些标签的渲染行为在各种浏览器中通常是相似的。

和flex有关的css属性：

#### 容器（Flex Container）属性

- `display`: flex 或 inline-flex
  - 将容器设置为Flex布局或内联Flex布局。
- `flex-direction`:
  - 定义项目（flex items）的排列方向。
  - 可选值：`row`（默认）、`row-reverse`、`column`、`column-reverse`。
- `flex-wrap`:
  - 定义项目是否应该在需要时换行。
  - 可选值：`nowrap`（默认）、`wrap`、`wrap-reverse`。
- `flex-flow`:
  - 是 `flex-direction` 和 `flex-wrap` 的简写形式。
- `justify-content`:
  - 定义项目在主轴上的对齐方式。
  - 可选值：`flex-start`、`flex-end`、`center`、`space-between`、`space-around`、`space-evenly`。
- `align-items`:
  - 定义项目在交叉轴上的对齐方式。
  - 可选值：`flex-start`、`flex-end`、`center`、`baseline`、`stretch`（默认）。
- `align-content`:
  - 定义多行项目在交叉轴上的对齐方式。
  - 可选值：`flex-start`、`flex-end`、`center`、`space-between`、`space-around`、`stretch`。

#### 项目（Flex Items）属性

- `order`:
  - 定义项目的排序顺序。
  - 默认值：`0`，数值越小越靠前。
- `flex-grow`:
  - 定义项目在容器中剩余空间的放大比例。
  - 默认值：`0`。
- `flex-shrink`:
  - 定义项目在容器空间不足时的缩小比例。
  - 默认值：`1`。
- `flex-basis`:
  - 定义项目在分配多余空间之前的默认大小。
  - 可选值：如 `auto`、`fill`、`content` 或具体尺寸。
- `flex`:
  - 是 `flex-grow`、`flex-shrink` 和 `flex-basis` 的简写形式。
  - 可选值：`flex-grow flex-shrink flex-basis`，如 `1 1 auto`。
- `align-self`:
  - 允许单个项目有与其他项目不同的交叉轴对齐方式。
  - 可选值：`auto`、`flex-start`、`flex-end`、`center`、`baseline`、`stretch`。
