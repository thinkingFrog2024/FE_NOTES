# JavaScript 代码规范

## 当读取 .js 文件时的特殊规则

### 代码质量要求
- 函数必须有完整的 JSDoc 注释
- 使用 ES6+ 语法特性
- 避免使用 var，优先使用 const

### 命名规范
- 变量和函数：camelCase
- 类：PascalCase
- 常量：UPPER_SNAKE_CASE

### 特殊注意事项
- 这个目录是学习笔记配套代码
- 代码需要有详细的中文注释
- 每个文件都应该可以独立运行演示
